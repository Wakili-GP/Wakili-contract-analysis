# ⚖️ AI Legal Auditor (المستشار القانوني الذكي)

نظام متطور يعتمد على الذكاء الاصطناعي (RAG Architecture) لتدقيق وتحليل العقود القانونية وفقاً للقانون المصري، باستخدام موديلات Llama 3.3 وتقنيات معالجة اللغة الطبيعية.

## 🚀 المميزات (Features)
* **تحليل متعدد الصيغ:** يدعم ملفات PDF و Word (DOCX).
* **دقة قانونية:** الردود مصممة لتناسب لغة المحاكم المصرية والقانون المدني.
* **استخراج ذكي:** تلخيص البنود، كشف المخاطر القانونية، وتحديد الالتزامات المالية.
* **دردشة تفاعلية:** إمكانية سؤال المستشار عن أي تفاصيل دقيقة داخل العقد.

## 🛠️ التقنيات المستخدمة (Tech Stack)
- **Language Model:** Llama-3.3-70b via Groq.
- **Frameworks:** LangChain, Streamlit.
- **Vector Database:** ChromaDB.
- **Embeddings:** HuggingFace (Local Model).
- **Backend:** FastAPI (Under Development).
